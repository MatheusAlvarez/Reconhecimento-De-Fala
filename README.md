## 💻 Sobre o projeto

Reconhecimento-De-Fala é um projeto criado para desenvolver o conhecimento na linguagem de programação Python

Um Programa que faça o reconhecimento da sua voz e escreva depois

---

## ⚙️ Funcionalidades

- [x] Gravar um áudio
- [x] Captar o áudio gravado pelo usuário
- [X] Escrever o que conseguiu captar através do áudio do usuário
